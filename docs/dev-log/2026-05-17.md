# 2026-05-17 实施记录：全面审查与 9 阶段改进

## 背景

对项目进行全面审查后，按 9 个阶段实施修复和改进：安全漏洞修复、Bug 修复、代码去重、无障碍与 UX 改进、性能优化、暗色模式、国际化 (i18n)、错题统计仪表盘和学习数据备份与恢复。

## 决策

- 暗色模式通过 CSS 自定义属性实现，不引入 CSS-in-JS 框架；`[data-theme="dark"]` 覆写约 50 个语义化颜色变量。
- 国际化通过 React Context + hook 实现，不引入 i18next 等运行时库；翻译文件为静态 TypeScript 对象，约 160 个消息键。
- 仪表盘图表用纯 CSS + SVG 实现（学科雷达图 SVG polygon，知识点/易错点柱形图 CSS），不引入 Chart.js 或 Recharts。
- 学习数据备份格式定义为 `StudyLibraryBackup` 接口（version: 1），导入支持三种合并策略（替换、合并优先导入、合并优先本地）。
- 安全修复：SSRF 防护补充 IPv6 保留地址检测；代理上游请求添加 120 秒超时；proxyClient 添加 30 秒默认超时；密钥脱敏扩展到 `x-goog-api-key` 和 `token=` 参数；移除 HTTP-only 服务上的无效 HSTS 头。
- 提取 `apiShared.ts` 消除 openaiClient/proxyClient 之间的元数据解析、上下文截断和历史构建重复代码。
- 发布归档时新增 `v1.3.0` 应用内新增功能向导；历史版本向导继续保留 `v1.2.0` 及更早版本。

## 用户可见变化

- 设置页新增主题切换（亮色/暗色/跟随系统），整个界面适配暗色主题。
- 已建立中英文 i18n 基础设施；当前语言设置主要控制模型回答和导出语言，界面文案后续逐步接入。
- 学习库新增"数据统计"入口，展示学科雷达图、知识点/易错点柱形图和统计卡片。
- 学习库新增"导出备份"和"导入备份"按钮，支持 JSON 格式全量数据备份和恢复，导入时可选替换或合并模式。
- 键盘焦点可见、模态面板焦点陷阱、减少动画设置支持。
- "清空学习库"按钮改为红色危险样式。
- 修复评测面板模型名不同步、OCR 图片缩放异常、对话加载后不滚动到底部等已知问题。

- 暗色模式下部分第三方渲染内容（KaTeX 公式）可能需要额外样式微调。
