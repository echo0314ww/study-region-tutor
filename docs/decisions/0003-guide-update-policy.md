# 0003 Guide Update Policy

## Status

Accepted

## Context

应用内有整体功能向导、本版本新增向导和历史版本向导回顾。用户可能从旧版本升级，也可能直接安装最新版。

## Decision

每次版本更新或用户可见变化都必须检查 `src/renderer/src/guides.ts`：

- 当前版本是否需要“本版本新增向导”。
- 历史版本向导回顾是否仍保留旧版本条目。
- 整体功能向导是否需要同步主流程变化。

旧版本新增向导条目不得删除。用户可感知的新入口、新流程、新配置或迁移提示应优先写入本版本新增向导。

## Consequences

- 新用户可通过历史版本向导补看旧功能。
- 升级用户可看到当前版本重点变化。
- 发布前清单和 `docs:check` 必须覆盖当前版本向导存在性。
